# Specular

General-purpose reflection utilities for Java.

## What's in the box

- **`TypeRef<T>`** — super-type token that captures generic types at compile time via an anonymous subclass.
- **`Types`** — helpers for resolving generic type parameters and method parameter types in a class hierarchy.

## Dependencies

- Apache Commons Lang 3 (for `TypeUtils`)
- JDK 25+
